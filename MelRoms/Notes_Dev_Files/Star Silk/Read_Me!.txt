Star Threader lets you link this python file to another.
This then allows MelRoms to lauch python files from any 
folder, not just from the Modules folder. Simply place
this in the Modules folder with any name you wish,
pick your program or game, and the program will lauch it
when opened from the launcher.

Or simply manually add it in the above list.
