# 🪐 MelRoms

MelRoms is a suite of programs I wasn't easily able to find and customize. I wanted specific things that no one made, so I did.

## ✨ Features
* **Simple Design:** Easily modify and add new modules / programs.
* **Custom Theming:** Supports dynamic JSON-based color schemes.
* **Optimized UI:** Designed for a high-performance, minimalist experience, though may not be perfect!
